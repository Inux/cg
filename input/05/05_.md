# Repetition

* Spiegelung
* Translation
* Rotation
* Skalierung

in 2D braucht man eine 3x3 Matrix. In 3D 4x4 eine Matrix.

Vektor 2D -> (x, y, w)

Skalarprodukt:
( 1 0 tx ) * (x)  =  x + tx
  0 1 ty   *  y      y + ty
  0 0 1    *  1      1

Assoziativgesetz:
R * (T * v) = (R * T) * v

Kommutativgesetz:
R * T != T * R
